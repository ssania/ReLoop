# ReLoop Database

## Tech Stack
- MongoDB Atlas (cloud database)
- Mongoose (schema modeling)
- AWS S3 (image storage)

## Collections
- users
- listings
- housingareas
- reviews
- housingreviews

## Setup Instructions
1. Clone the repo
2. cd into database/
3. Run npm install
4. Create a .env file with your MONGODB_URI
5. Run node seed.js
